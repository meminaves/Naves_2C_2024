◆ manejarLeds()

static void manejarLeds ( )
static

Controla el encendido de los LEDs según la distancia medida.

Si la distancia es menor a 10 cm, todos los LEDs están apagados. Si la distancia está entre 10 y 20 cm, se enciende el LED_1. Si la distancia está entre 20 y 30 cm, se encienden los LEDs 1 y 2. Si la distancia es mayor a 30 cm, se encienden todos los LEDs.

static void taskMostrarMedida ( void * pvParameter)
static

Tarea que muestra la medida en la pantalla y maneja los LEDs.

Si el sistema está encendido (ON), se muestra la medida en la pantalla y se controlan los LEDs según la distancia medida. Si el sistema está en modo HOLD, la medida no se actualiza.
